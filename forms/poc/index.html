Setup steps:
1. Create a blank Google Sheet
2. Go to script.google.com → New Project
3. Paste the contents of web-app.gs into Code.gs
4. Replace YOUR_SHEET_ID_HERE with your Sheet ID (the long string between /d/ and /edit in the sheet URL)
5. Deploy → New deployment → Web app → Execute as: Me, Access: Anyone
6. Paste the Web App URL into the field above
7. Fill out the form and submit → check the sheet for a new row
